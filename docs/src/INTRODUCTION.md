# Introduction to Kairos-rs

Welcome to the documentation for **Kairos-rs**, a high-performance, AI-powered API gateway written in Rust.

Kairos-rs is designed to be:
- **Fast**: Built on Actix-web and optimized for low latency.
- **Intelligent**: Leveraging LLMs for smart routing and request processing.
- **Flexible**: Supporting multiple protocols (HTTP, WebSocket, FTP, DNS).

Navigate through the guides to learn more about specific features and integrations.
